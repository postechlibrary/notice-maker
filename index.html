📖 처음 사용하세요? 사용 방법 보기
  1. 카테고리 선택 — 공지 종류(일반 안내/긴급 등)를 고르면 본문 템플릿이 자동으로 채워집니다.
  2. 제목 입력 — 한국어 제목을 쓰고, 「제안」 버튼을 누르면 영문 제목이 자동으로 채워집니다.
  3. 부서·날짜 입력 — 담당 부서명과 공지 날짜를 입력하세요.
  4. 본문 작성 — 아래 편집창을 클릭하면 커서가 생깁니다. 위의 툴바(B·I·U·H2·목록 등)로 서식을 적용하세요.
  5. 검토 — 오른쪽 「가이드라인」 탭에서 규칙 통과 여부를 확인합니다. 빨간 항목은 수정이 필요합니다.
  6. 복사 — 하단 준수율이 충분히 오르면 「HTML 복사」 버튼이 활성화됩니다. 눌러서 WordPress에 붙여 넣으세요.
